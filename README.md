### Как запустить
1. Скачиваем
2. В консоли пишем ***npm i***
3. Запускаем либо в режиме разработки - ***npm run dev***
4. Либо билдим - ***npm run build***, и запускаем - ***npm run start***
5. Запускается на localhost:3000 (в браузере переходим на - http://localhost:3000/)
